# Concepts

## Evidence Map Workflow

The Evidence Map workflow is the inspectable substrate beneath a polished artifact. It records sources, claims, assumptions, calculations, conflicts, and review status before the artifact travels.

## Source Packet

A source packet is the controlled inventory of messy input material. It separates current sources from stale sources, facts from assumptions, and conflicts from resolved claims.

## Evidence Map

An evidence map links a claim, slide, cell, chart, paragraph, or table to the source IDs that support it.

## Hostile Review

A hostile review is a verification pass that assumes the artifact is wrong until it proves otherwise. It enumerates issues. It does not rewrite the artifact.

## Workbook Doctor

Workbook Doctor is the Excel-specific review layer: raw data, assumptions, calculations, checks, formula consistency, hardcodes, and stale date ranges.

## Deck Architect

Deck Architect is the PowerPoint-specific structure layer: audience, decision, narrative spine, slide claims, speaker-note evidence, and unresolved questions.
