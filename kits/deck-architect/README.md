# Deck Architect Kit

Use this kit to define a trustworthy deck before rendering slides.

## Output

- `deck-spec.md`
- `slide-claim-map.json`
- speaker-note evidence template

## Required Fields

For every slide:

- slide number
- claim headline
- decision role
- source IDs
- data date range
- assumptions
- open questions
- review status

## Speaker Notes Template

```text
Claim:
Source IDs:
Calculation or transformation:
Assumptions:
Needs review:
```

## Rule

Slide headlines should be traceable claims, not topics.
