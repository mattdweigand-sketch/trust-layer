# Evidence Map Kit

Use this kit to connect artifact content to the source packet.

## Evidence Targets

- slide claim
- speaker note
- workbook cell
- workbook chart
- document paragraph
- table row
- metric

## Required Fields

- target ID
- artifact location
- claim or value
- source IDs
- source date
- owner
- transformation
- assumption flag
- confidence
- review status

## Review Status Values

- `unreviewed`
- `reviewed`
- `needs_human`
- `blocked`
