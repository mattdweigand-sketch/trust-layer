# Prompt 02: File Specification

You are creating the file specification from an approved source packet.

Do not render the final artifact yet.

If the target is a deck, produce:

- audience
- decision context
- one-sentence narrative spine
- slide list
- claim headline for each slide
- evidence source IDs per claim
- visuals or charts required
- assumptions
- open questions
- speaker-note evidence requirements

If the target is a workbook, produce:

- workbook purpose
- user
- tab architecture
- raw data fields
- assumption list
- calculation flow
- output metrics
- required checks
- formula and hardcode rules
- scenario inputs

Mark any unsupported claim or number as `needs_human`.
