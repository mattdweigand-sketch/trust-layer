# Prompt 05: Evidence Map

Create an evidence map for this artifact.

Map every material claim, number, chart, table, workbook cell, or document paragraph to the source packet.

For each item, return:

- target ID
- artifact location
- claim or value
- source IDs
- source date
- owner
- transformation or calculation
- assumption flag
- confidence
- review status

Use `needs_human` when the claim is consequential and not fully supported.

Do not rewrite the artifact.
