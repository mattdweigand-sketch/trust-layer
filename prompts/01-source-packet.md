# Prompt 01: Source Packet

You are preparing a source packet before any Office artifact is created.

Do not write slides, formulas, executive prose, or recommendations yet.

Inspect the provided source set and produce:

1. Source inventory
2. Source status labels
3. Authority ranking by fact class
4. Fact list
5. Assumption list
6. Estimate list
7. Conflict log
8. Missing context
9. Human review decisions required

Use source IDs `S01`, `S02`, `S03`, etc.

For every source, capture:

- source ID
- title
- owner
- date
- file type
- status
- authority
- currentness
- notes

Stop if a conflict affects a decision-critical claim or number.
